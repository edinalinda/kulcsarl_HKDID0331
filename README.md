# hipobarias_vizsgalat_terminal.py feladathoz leírás

Ez az önálló Python script parancssoros felületet biztosít a hipobariás számításokhoz. A felhasználó a terminálban adhatja meg a magasságot, mértékegységet, feladat nehézségét és HSI minőséget, majd a program kiírja a reakcióidőt, hibaarányt és hipobariás szintet.

#Használata
1. (Ajánlott) Aktiváld a Python virtuális környezetet:

```zsh
source .venv/bin/activate
```

2. Futtasd a scriptet:

```zsh
python hipobarias_vizsgalat_terminal.py
```

3. Kövesd a terminálban megjelenő utasításokat, add meg a szükséges adatokat. Az eredmények szövegesen jelennek meg.

# Főbb jellemzők
- Interaktív adatbekérés.
- Eredmények egyértelműen megjelennek a terminálban.

# Követelmények
- Python 3.x
